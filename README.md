# Simple Python Calculator

A basic command-line calculator that performs arithmetic operations with an interactive interface.

## Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with zero division error handling)
- 🔄 Continuous operation until user quits
- 🚪 Quit option using 'q'

## How to Use
1. Run the program: `python calculator.py`
2. Enter an operation symbol (+, -, *, /) or 'q' to quit
3. Enter first number
4. Enter second number
5. View the result
6. Repeat steps 2-5 or quit with 'q'
